# Project - Zainea Bogdan - Phone Book

## Goal

The purpose of this exercise is to create a web page for managing a phonebook using concepts from the first three labs (**HTML**, **CSS**, **JavaScript**).

---

## Requirements

1. **Page Structure**  
   Create a web page with the following structure:

   - A **title**
   - A **form** with three fields and an **Add Person** button
   - A **table**  
     _(Presentation suggestion: Image 1)_  
     **– 1p**

2. **Table Header and Footer**  
   The table must include a header and a footer, with the footer displaying the total number of people in the phonebook.  
   **– 1p**

3. **Layout and Styling**  
   Use spacing, margins, borders, and other visual styling elements.  
   **– 1p**

4. **Attributes**  
   All important elements on the web page must have assigned **id** and **class** attributes.  
   **– 1p**

5. **Interactivity**  
   Add **hover** and **focus** effects for various elements of the page.  
   **– 1p**

6. **Add Functionality**  
   Define a function **`addPerson()`** that adds new rows to the table and increases the total number of people.  
   **– 1p**

7. **Delete Functionality**  
   Add a **Delete Person** button that removes a row from the table and decreases the total number of people.  
   **– 1p**

8. **Form Reset**  
   Automatically clear the form after data is submitted.  
   **– 1p**

9. **CSS Animation**  
   Use **@keyframes** to create a loading animation effect when a new person is added.  
   **– 1p**

10. **Theme Switcher**  
    Add a **dark/light theme** option for the created page.  
    **– 1p**
